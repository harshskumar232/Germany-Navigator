# Germany Navigator (V3)

This zip includes a complete, clean UI/UX refresh:
- One global CSS (`gn.css`) and JS (`gn.js`) for reliability.
- No checkmarks, no editable tables: info-first, source-first.
- Local SVG hero image in `/assets` so it looks good without external hosting.
- Pages: Home, Start Here (A→Z), Admission, Visa, Before Departure, After Landing, City Guide, German Test (placeholder), Social.

## GitHub Pages
Deploy with GitHub Pages (Settings → Pages → Branch: main / root).  
Custom domain must be a *real* domain like `germanynavigator.com` or `www.germanynavigator.com` (not just `germanynavigator`).
